Overview

Note: You should not use Collection directly. It will be automatically created by things like {Joosy.Resource.Generic#map} or {Joosy.Resource.REST#find}.

Basic collection of Resources. Turns JSON array into array of Resources and manages them.

Examples:

Basic sample 

class R extends Joosy.Resource.Generic
  @entity 'r'
  
collection = new Joosy.Resource.Collection(R)
  
collection.reset [{foo: 'bar'}, {foo: 'baz'}]
collection.each (resource) ->
  resource('foo')
